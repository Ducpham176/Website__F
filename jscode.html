Một Số Sự Kiên Js
  • 🎯click: Sự kiện này xảy ra khi người dùng nhấp chuột vào một phần tử.
  • 🎯submit: Sự kiện này xảy ra khi người dùng gửi một biểu mẫu (form).
  • 🎯keydown: Sự kiện này xảy ra khi người dùng nhấn một phím trên bàn phím.
  • 🎯keyup: Sự kiện này xảy ra khi người dùng thả một phím đã nhấn trên bàn phím.
  • 🎯load: Sự kiện này xảy ra khi một tài nguyên (như hình ảnh ...)được tải hoàn tất.
  • 🎯scroll: Sự kiện này xảy ra khi người dùng cuộn trang web.
  • 🎯change: Sự kiện này xảy ra khi giá trị của một phần tử đã được thay đổi.
  • 🎯focus: Sự kiện này xảy ra khi một phần tử được tập trung phần tử đang hoạt động.
  • 🎯input: Sự kiện này xảy ra khi người dùng nhập dữ liệu vào (như hộp văn bản).

    • document.querySelectorAll()

    câu lệnh này sẽ được dùng khi muốn tìm nhiều phần tử
    🔥ví dụ 1: ứng dụng trong website: bạn có rất nhiều sản phẩm
    bạn không thể nào đặt 1 sản phẩm là tên 1 class rồi đi gọi sự kiện
    lắng nghe lên từng class một/ có 100 sản phẩm thì sao.

    🔥Ví dụ 2: có 50 class sản phẩm tên 'card'

    Lúc này nó sẽ phát huy tác dụng: bạn chỉ cần querySelectorAll('.card')

    và dùng vòng for để duyệt qua:

    ✅: for(let i = 0; i < document.querySelectorAll('.card').length; i++)
    Sau đó xử lí thôi !

    document.querySelectorAll('.card')[i].addEventListener() ᴳᵒᵈ🚵
    🔴( nhớ viết trong ngoặc {} )

    Bên đây mình sẽ chia sẽ một số ví dụ ứng dụng 💯



    const cardElements = document.querySelectorAll('.item');
    for (let i = 0; i < cardElements.length; i++) {
    const card = cardElements[i];
    card.addEventListener('click', () => {
    for (let j = 0; j < cardElements.length; j++) { if (j !== i) { cardElements[j].style.flexGrow = 1;
    }
    }
    card.style.flexGrow = 5;
    });
    }


    🐋 Đầu tiên đặt tất cả các class item là cardElements ( để cho ngắn )
    Tiếp theo
    🐋 Trong vòng For mình bắt sự kiện click ở từng phần tử.
    🐋 Vòng For thứ 2 mình xét xem phần tử có khác phần tử đang chọn
    🌈 Nếu khác thì fexGrow = 1 => Tức trạng thái mặc định.

    🌈 Nếu chưa thì xét là 5


    ⛄forEach và for là hai cách khác nhau để lặp qua các phần tử trong một mảng hoặc một danh sách các phần tử.


    Tượng tự ví dụ trên nhưng dùng forEach

    const cardElements = document.querySelectorAll('.item'); cardElements.forEach((card, i) => {
    card.addEventListener('click', () => {
    cardElements.forEach((otherCard, j) => {
    if (j !== i) {
    otherCard.style.flexGrow = 1;
    }
    });
    card.style.flexGrow = 5;
    });
    });
